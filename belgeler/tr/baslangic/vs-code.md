# VS Code Entegrasyonu

Vale'i kurup Türkçe dil paketini yükledikten sonra, VS Code'da yazarken gerçek zamanlı uyarılar alabilirsiniz.

## Kurulum

1. VS Code'da **"Vale VSCode"** eklentisini yükleyin
2. Ayarlardan `vale.configPath` değerini belirleyin:
    - macOS/Linux: `~/.config/vale/.vale.ini`
    - Windows: `%USERPROFILE%\.vale\.vale.ini`

## Sonuç

Artık yazarken hatalar anında altı çizili olarak görünür. Üzerine geldiğinizde düzeltme önerisi çıkar.

!!! tip "İpucu"
    VS Code entegrasyonu için önce [Kurulum](kurulum.md) adımlarını tamamlamış olmanız gerekir.
